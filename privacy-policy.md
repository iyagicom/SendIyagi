Privacy Policy for SendIyagi
Last updated: February 26, 2026

Overview
SendIyagi is a peer-to-peer (P2P) communication tool that allows users to send files and exchange messages directly between devices.

Features
SendIyagi provides the following features:
- Send files to another device
- Receive files from another device
- Real-time chat communication
- Simple help guide

Data Collection
SendIyagi does not collect, store, or transmit any personal data to external servers.

Local Network Communication
SendIyagi uses peer-to-peer (P2P) connections to transfer files and messages directly between devices. Data is exchanged only between the connected devices and is not routed through any external server.

Local Data Storage
Files received through SendIyagi are saved locally on the user's device. Chat messages may also be stored locally during the session.

No Central Server
SendIyagi does not use a central server. All communication occurs directly between users.

Internet Usage
SendIyagi may use local network or internet connections only for direct peer-to-peer communication between devices.

User Control
Users have full control over the files they send or receive and can delete files or chat history at any time.
