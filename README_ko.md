# SendIyagi v1.4.2

Windows용 **초경량 고속 P2P 파일 전송 + 채팅 앱**

클라우드 없이, 계정 없이, 파일 크기 제한 없이  
PC ↔ PC로 직접 전송하는 구조이다.

---

## ✨ 주요 기능

### 📁 파일 전송 (P2P)
- P2P 직접 전송 (TCP 기반, 중계 서버 없음)
- 드래그 & 드롭으로 즉시 전송 목록 추가
- 폴더 전체 전송 (구조 그대로 유지)
- 다중 수신자 지원 (여러 클라이언트 동시 다운로드)
- 실시간 전송 속도 표시
- 연결 실패 시 자동 재시도

---

### 🌐 HTTP 전송
- HTTP 업로드 서버 (브라우저에서도 업로드 가능)
- 청크 기반 업로드 + 재개 지원
- 업로드 키(비밀번호) 설정 가능

---

### 🌍 네트워크 기능
- LAN 자동 탐색 (같은 네트워크 장치 자동 감지)
- 원클릭 연결 (탐색된 장치 클릭으로 IP 자동 입력)
- UPnP 포트 자동 개방 (인터넷 전송 지원)
- 외부 IP 표시 및 복사 기능

---

### 💬 채팅 기능
- 텍스트 + 이미지 실시간 채팅
- 최대 20명 동시 접속 지원
- 읽지 않은 메시지 배지 표시
- 채팅 배경색 커스터마이징

---

### ⚙ 일반 기능
- 한글 / 영어 자동 UI 전환 (OS 로케일 기반)

---

## 🚀 사용 방법

### 📤 보내는 쪽 (Server 탭)
1. 파일 또는 폴더를 드래그 앤 드롭
2. 포트 확인 (기본: 4567)
3. “대기 시작” 클릭
4. 로컬 IP 또는 외부 IP 공유

---

### 📥 받는 쪽 (Client 탭)
1. 서버 IP + 포트 입력 (또는 LAN 자동 탐색 클릭)
2. 저장 폴더 선택
3. Connect 클릭
4. 다운로드할 파일 선택 후 받기

---

### 🌐 HTTP 전송
- Server 탭: 저장 폴더 + 업로드 키 설정 후 HTTP 서버 자동 실행
- Client 탭: URL 입력 후 파일 업로드 (드래그 지원)

---

### 💬 채팅
- 연결 후 Chat 탭으로 이동
- 메시지 / 이미지 실시간 전송 가능
